<h1>Gerenciador de Mesas de RPG!</h1>
<h2> Tecnologias: </h2>
  Back End: NodeJS <br>
  Front End: HTML5, Css e JS 

<h2> Sobre: </h2>

<p>
  <b>Ser um mestre de RPG não é para todos!</b> Todo DM veterano sabe que suas atribuições estão muito além de apenas mestrar uma mesa. Se afogar em diversas demandas como: 
  <i>Produzir o game design, escrever a narrativa e gerenciamento de evento e pessoas</i>. <b>São um medo genuíno e muito real!</b>
</p>
<p>
  Por essa razão, Estou criando o Nosso <b>Grenciador de Mesas</b> é uma mão na roda para os mestres,
  dos calouros aos veteranos. Permitindo você a <b>CRIAR SUAS MESAS</b> e <b>GERENCIAR</b> os diversos <b>TIPOS de ATIVIDADES</b>, <b>JOGADORES</b> e <b>SISTEMAS</b>.
</p>

Gosto muito de mestrar sessões de RPG de Mesa, então decidi misturar minhas duas paixões em um projeto muito ambicioso!
